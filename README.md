# Personal Portfolio Website - Jeren Aubrey M. Orbeta

A fully responsive personal portfolio website built to showcase my projects, skills, and experience as an IT student. This project was built from scratch using HTML, CSS, and JavaScript.

This project fulfills all requirements for the "Resume/Portfolio Website" assignment, including:
* Sticky navigation
* A parallax scrolling section
* Embedded YouTube videos
* A fully responsive layout (mobile, tablet, and desktop)
* A 2-column contact section

## How to Run Locally

To view this project on your local machine:

1.  Clone this repository:
    ```sh
    git clone https://github.com/jerenaubreyorbeta/MyPortfolio_Orbeta-Jeren-Aubrey-M.
    ```
2.  Navigate into the new project folder:
    ```sh
    cd portfolio-website
    ```
3.  Double-click the `index.html` file to open it in your default web browser.
